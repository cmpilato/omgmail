# omgmail

Gmail IMAP gateway.

## Package name

- Distribution/package name: `OMGmail`
- Import/module name: `omgmail`

## Install

From the repository root:

```bash
python -m pip install -e .
```

## Run

Use the installed console script:

```bash
omgmail < /path/to/message.eml
```

or invoke as a module:

```bash
python -m omgmail < /path/to/message.eml
```
